//APPLICATION INFORMATION//
  • Purpose: This app allows you to track and export missing size data across various product types by gender. Data is loaded automatically from Mens and Womens assortment lists for S2025.
    •
  • Required Fields: To add an entry, you must fill out Store Name and select Gender, Product Name, Product Code, and at least one Size.
    •
  • Size Grid: Size options dynamically adjust based on the automatically determined product type. Sizes are sorted logically (e.g., XS, S, M, L or numerically) when an entry is added.
    •
  • Entry List: Entries are sorted by gender and then grouped by product type.
    •
  • Size Limitations: Plus sizes are currently not supported.
    •
  • Mobile Use: This app is mobile-friendly and supports touch selection for size buttons.
    •
