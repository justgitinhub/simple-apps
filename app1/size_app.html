// INSTRUCTIONS //
  • Required Fields: To add an entry, you must fill out Store Name and select Gender, Product Name, Product Code, and at least one Size. (Adding a note to an entry is optional.)
    •
  • Save Entry: Click the '+ Add Entry' button. The entry will appear in the list below.
    •
  • Deleting an Entry: Click the red circle icon on the top right of the entry to remove it.
    •
  • Downloading List: When all entries are added, click the 'Download List .XLSX' button. [[ Mobile - If a pop-up box appears on mobile, click the "Download" button in that pop-up. ]]
    •
  • Desktop Downloads: Files are saved to your "Downloads" folder or your specified default download location.
    •
  • Mobile Downloads: Open your device's "Files" app and check the "Downloads" folder.
    •

     If files downloaded from our app (via Safari) aren't in your Files app "Downloads" folder:

    1. Quick Fix: Manually Save from Safari
  • In Safari, tap the "Aa" or download arrow (â†“) in the address bar.

  • Tap Downloads.

    •

    •
  • Choose "Save to Files."
    •
  • IMPORTANT: Select where to save it (e.g., "On My iPhone" > "Downloads" or an iCloud Drive folder). Tap Save.
    •

    2. Check Safari's Default Save Spot:
  • Go to iPhone Settings > Safari > Downloads.
    •
  • Note the location:
    • "On My iPhone": Check Files app > Browse > On My iPhone > Downloads.
      •
    • "iCloud Drive": Check Files app > Browse > iCloud Drive > Downloads.
      • (Ensure iCloud Drive is ON: Settings > Your Name > iCloud > iCloud Drive).
        •
  • Change this setting if you prefer a different default location for future downloads.
    •

    3. Search in Files App:
  • Open Files app.
    •
  • Tap Browse (bottom right).
    •
  • Use the Search bar at the top (try file name or .xlsx).
    •

    Still Stuck with Downloads?
  • Restart your iPhone.
    •
  • Ensure your iOS is updated (Settings > General > Software Update).

    // APPLICATION INFORMATION //
  • Purpose: This app allows you to track and export missing size data for products, categorized by gender.
    •
  • Dynamic Size Grid: Size options automatically update based on the selected product. Sizes are sorted logically (e.g., XS, S, M, L, or numerically).
    •
  • Entry List Organization: Entries are sorted first by gender, then grouped by product name.
    •
  • Downloaded Format: Your entry list will be saved as an Excel file (.xlsx). The filename will follow the format: 123_Store_Name_MissingSizes_[Date]. (e.g., 123_Store_MissingSizes_2025-05-1)
    •
  • Size Limitations: Plus sizes are currently not supported.
    •
  • Mobile Use: This app is mobile-friendly and supports touch selection for size buttons.
    •
  • Data Source: Spring/Summer 2025 (S2025) Men's and Women's assortment data is automatically loaded.
    •
